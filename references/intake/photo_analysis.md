# Photo Analysis

Use only when user provides or asks about photos.

Assess non-medically:
- cambiamento visivo
- distribuzione apparente del grasso
- postura/posa/luce che possono falsare confronto
- proporzioni utili per monitoraggio

Output:
```json
{
  "visual_change": "",
  "confidence": "low|medium|high",
  "tracking_notes": [],
  "recommendations": []
}
```

Do not diagnose, estimate exact body fat, or infer health conditions.
