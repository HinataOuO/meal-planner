# Nutrition Assessment

Analyze current intake before planning.

Collect:
- pasti tipici
- orari
- snack
- bevande/alcol
- pasti fuori
- episodi di fame o abbuffate
- aderenza a piani precedenti

Output:
```json
{
  "current_pattern": "",
  "adherence_barriers": [],
  "easy_wins": []
}
```

Do not generate a full plan from this file alone.
