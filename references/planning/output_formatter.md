# Output Formatter

Use concise structure:

1. Calories and macros
2. Meal plan table
3. Meal prep steps
4. Storage notes
5. Shopping list, if requested
6. Check-in metrics to report next week

For adaptations:
1. What changed
2. New calories/macros or meal swaps
3. Why in one short line
4. Next check-in target

Keep formatting clean. Avoid long theory sections.
